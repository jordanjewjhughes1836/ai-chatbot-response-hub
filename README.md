## Usage

Once the app is running, open it in a browser and start chatting from the interface.

Typical workflow:

1. Enter a message in the chat window.
2. Let the backend process the text with NLP logic.
3. Review the generated response, summary, or sentiment output where supported.
4. Continue the conversation while the system keeps track of chat history.
